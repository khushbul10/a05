Question-1: What are the different ways to select an element in the DOM?

Answer: There are multiple ways to select an element in the DOM. Some of them are:
  1. By ID-getElementById()
  2. By Class Name-getElementsByClassName()
  3. By Tag Name-getElementsByTagName()
  4. By CSS Selector (First Match)-querySelector()
  5. By CSS Selector (All Matches)-querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1. innerHTML: Retrieves or sets the HTML content inside an element.
2. innerText: Retrieves or sets the visible text content of an element.
3. textContent: Retrieves or sets the text content of an element including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events on its child elements, including those added dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism in the DOM where an event starts at the target element and propagates up through its parent elements until it reaches the root.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create an Element using document.createElement().
2. Add an Element using append(), appendChild(), prepend().
3. Remove an Element using removeChild() or remove().
